# 🚀 KnexByte Editor

**KnexByte Editor** is an elite, minimalist, and installable PWA (Progressive Web App) designed for professionals and developers. It offers a seamless rich-text editing experience with a focus on speed, aesthetics, and portability.

## ✨ Features

- **Standard Modern Interface**: A smooth dark/light mode experience with a premium layout.
- **Rich Text Suite**: Complete control over bold, italics, underlining, alignment, and lists.
- **Advanced Typography**: High-performance font engine featuring **Calibri** as the default, with a comprehensive list of professional fonts and precise size controls (8px to 96px).
- **Developer Ready**: Built-in support for code blocks with syntax highlighting and table insertion.
- **Document Management**: Manage multiple local documents via a clean, intuitive sidebar.
- **Offline Capable (PWA)**: Install it as a desktop or mobile application and work offline.
- **One-Click Export**: Export your documents to high-quality HTML files instantly.

## 🤖 Upcoming AI Integration (Roadmap)

## 🛠 Tech Stack

- **Core**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Modern CSS with CSS Variables
- **PWA Engine**: `vite-plugin-pwa`
- **Icons**: Lucide React

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/shoaib171/KnexByte-PWA-editor_export-to-pdf-file.git
   ```
2. Navigate to the project directory:
   ```bash
   cd knexbyte-text-editor
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production
To create a production-ready build:
```bash
npm run build
```
